# Bank Customer Transactions Analysis

## Project Overview

This project analyses bank customer transactions using Python and data processing techniques.

The goal of the project is to explore and analyse financial transaction data by applying common database operations such as filtering, aggregation, and sorting.

The project was developed as part of an academic assignment focused on understanding data analysis and database-style queries using Python.

---

## Dataset

The dataset contains information about bank customers and their transactions, including:

- Customer ID
- Customer Name
- Transaction Amount
- Transaction Type
- Date

---

## Project Tasks

The project performs three main data analysis tasks.

### 1. Filtering

Transactions greater than **$10,000** are extracted from the dataset.

This helps identify high-value financial transactions.

---

### 2. Aggregation

The program calculates the **total transaction value for each customer**.

This is done using grouping operations similar to SQL `GROUP BY`.

---

### 3. Sorting

Customers are sorted based on their **total transaction amount**, showing the customers with the highest transaction values first.
